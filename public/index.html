LTV Korean Project

Currently a WIP. I'm creating this project to try to help other Korean learners who are in the same place as me, who like LOONA and want to try to learn Korean through the amount of subtitled content that LOONA has. Most of the translations come from Naver dictionary and from the LTVs themselves, or from my own experience, but if you feel they are wrong please feel free to correct me! You can post an issue at the icon below if you have a better translation or if you have any suggestions with the site.

There still is a lot to be done, so if you would like to help me translate please let me know! You can get in touch with me at the link below, or find my discord as razgrizaces#6969.
